# Large-Scale-Distributed-Clustering-of-NYC-DOB-Permit-Data
Implemented large scale distributed clustering to analyze construction and renewal hotspots in NYC building permit data of 3.9 Million rows from the years 1991-2025

# Dataset:
https://data.cityofnewyork.us/Housing-Development/DOB-Permit-Issuance/ipu4-2q9a/about_data


## Methods:
1. K-Means with MPI for distributed clustering
2. DBSCAN with n_job=20 for hotspot detection
